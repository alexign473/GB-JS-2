# Eshop-Cart-Vue-NodeJS
Проект по курсу Geekbrains Javascript 2

## Installation

Clone this repo and npm install.

```bash
npm i
```
## Usage

### Development build

```bash
npm run dev
```
### Development watch

```bash
npm run watch
```

### Development server

```bash
npm run start
```

You can view the development server at `localhost:3000`.

## Features

-Webpack
-Vue
-NodeJS
